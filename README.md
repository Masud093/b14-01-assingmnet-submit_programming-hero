# DevConf Landing Page

A simple responsive landing page project for a developer conference event.

## Project Structure
- index.html — main landing page
- join-team.html — separate placeholder page for the Join Our Team CTA
- style.css — shared styling for the site
- assets/ — images and static assets

## Preview
Open index.html in a browser to view the main page.
